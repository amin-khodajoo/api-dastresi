# API سایت دسترسی

این API مربوط به سایت **دسترسی** است که توسط خودم طراحی و نوشته شده است.  
فایل JSON حاوی داده‌ها و اطلاعات مورد استفاده در پروژه است و به عنوان منبع داده برای بخش‌های داینامیک سایت عمل می‌کند.
